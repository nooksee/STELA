Section-by-section authoring protocol:
- Write Section 3.1. Run the gate. Stop on mismatch. Freeze 3.1.

- Write Section 3.3. Lock scope and allowlist. Freeze 3.3.
- Write Section 3.4 in order (3.4.1 through 3.4.5). Freeze each subsection when complete.
- Do not revise earlier sections to justify later work. If scope changes, end the thread and draft a new DP.

- If a completed DP subsection changes later, append an AMENDMENT block in RESULTS; otherwise DP is FAIL.

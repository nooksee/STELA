# LLMS Bundle Manifest

## Purpose
Define scope-based context bundles for llms consumers.

## Ops Bundle
- `docs/MANUAL.md` - Command mechanics and top commands.
- `docs/MAP.md` - Wayfinding map.
- `ops/bin/open` - Session prompt generator.
- `ops/bin/dump` - State capture tool.
- `ops/bin/llms` - Context bundler.
- `ops/bin/map` - MAP auto block maintainer.
- `ops/bin/context` - Context snapshot assembler.
- `ops/bin/prune` - Archive hygiene.
- `tools/lint/context.sh` - Context lint.
- `tools/lint/llms.sh` - Bundle parity lint.
- `tools/lint/truth.sh` - Canon spelling lint.
- `docs/ops/specs/binaries/llms.md` - llms spec.
- `docs/ops/specs/binaries/map.md` - map spec.
- `docs/ops/specs/binaries/context.md` - context spec.
- `docs/ops/specs/binaries/dump.md` - dump spec.
- `docs/ops/specs/binaries/project.md` - project spec.

## Governance Bundle
- `PoT.md` - Policy of Truth (constitution, staffing, jurisdiction, enforcement).
- `SoP.md` - History ledger and state of play.
- `TASK.md` - Active thread, DP contract, and work log.
- `llms.txt` - Discovery entry point.
- `docs/GOVERNANCE.md` - Project governance and non-negotiables.
- `docs/MAP.md` - Wayfinding map.
- `ops/lib/manifests/CONTEXT.md` - Required context checklist.
- `ops/lib/manifests/LLMS.md` - Scope bundle definitions.
